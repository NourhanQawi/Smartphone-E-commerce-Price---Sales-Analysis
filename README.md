# Smartphone E-commerce Price & Sales Analysis

## Project Overview
This project analyzes smartphone products from an e-commerce dataset to understand:
- Pricing differences and discount patterns
- Sales volume distribution
- Brand performance
- Customer ratings vs price levels

The goal is to provide business insights that help decision-makers optimize pricing and product strategy.

## Dataset Description
The dataset contains smartphone listings with the following key fields:
- Product price & original price
- Discount difference
- Brand
- Star rating & number of reviews
- Sales volume
- Availability & delivery
- Price level classification (Low / Mid / High)

Total products analyzed: **319**

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning & Transformation
- Dashboard Design

## Key Insights
- High-priced phones have the highest average ratings.
- Samsung and Apple dominate sales volume.
- Low and mid price segments contain the largest number of products.
- Discounts significantly impact sales volume in mid-range phones.

## Dashboard
The Excel dashboard summarizes:
- Total sales volume
- Average ratings by price level
- Brand performance
- Product count distribution
